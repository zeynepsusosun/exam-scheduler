# 📅 Exam Scheduler

**Exam Scheduler** is an exam scheduling application built with **React + Vite**.  
It helps automatically plan exams, create suitable time slots, and generate optimized outputs in Excel format.

---

## 🚀 Features

- 📋 Automatic exam scheduling
- 📁 Optimized Excel output
- ⚡️ Fast and modern **React + Vite** frontend
- 🧩 Flexible structure, easy to extend

---

## 🛠️ Technologies

This project is built using:

- **React** — User interface
- **Vite** — Fast development and build tool
- **JavaScript / CSS** — Application language and styles
